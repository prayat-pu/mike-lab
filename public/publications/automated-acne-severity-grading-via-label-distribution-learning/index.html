Automated Acne Severity Grading via Label Distribution Learning

Suppawong Tuarob, Elena Rostova, Marcus Vance

View Publisher (DOI) Download PDF GitHub / Code

Abstract

This research addresses the inherent subjectivity in clinical acne grading by engineering a Label Distribution Learning (LDL) framework. Unlike single-label systems, our model captures the ambiguity between severity levels, providing dermatologists with a probabilistic distribution that generalizes across diverse clinical datasets and lighting conditions.
